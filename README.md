# DataCollect Pro

Application professionnelle de collecte et d'analyse descriptive de données commerciales, développée avec Python et Streamlit.

## Fonctionnalités

- Formulaire structuré de collecte de données entreprises
- Stockage automatique en CSV et Excel
- Analyse descriptive avec graphiques interactifs
- Export des données en CSV et Excel
- Interface professionnelle et épurée

## Technologies utilisées

- Python 3.x
- Streamlit
- Pandas
- Plotly
- OpenPyXL
- XlsxWriter

## Installation locale

1. Cloner le dépôt :
   git clone https://github.com/VOTRE_NOM/datacollect-pro.git

2. Créer un environnement virtuel :
   python -m venv venv
   venv\Scripts\activate.bat

3. Installer les dépendances :
   pip install -r requirements.txt

4. Lancer l'application :
   streamlit run app.py

## Utilisation

- **Accueil** : Tableau de bord avec les indicateurs clés
- **Formulaire** : Saisie des données entreprises
- **Analyse** : Visualisation et statistiques descriptives
- **Données** : Tableau complet avec export

## Déploiement

Application déployée sur Streamlit Cloud.
